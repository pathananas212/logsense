# 🔍 LogSense — AI Log File Analyzer

Paste any log file → Get a plain-English diagnosis + exact fix in seconds.

---

## 🚀 Quick Setup (5 Minutes)

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the app
```bash
streamlit run app.py
```

### 3. Open in browser
Go to: http://localhost:8501

---

## 🔑 You Need
- An **Groq API key** 
- Python 3.8+

---

## ✅ Supported Log Types
- 🐍 Python Tracebacks & Errors
- 🌐 Apache / Nginx Server Logs
- 🐧 Linux System & Kernel Logs
- ⬡ Node.js Runtime Errors
- 📄 Any Custom Log Format

---

## 💡 Features
- Paste or upload log files (.log, .txt, .out, .err)
- AI-powered plain English explanation
- Severity detection (Critical / High / Medium / Low)
- Root cause analysis
- Step-by-step fix instructions
- Prevention tips
- Downloadable report
